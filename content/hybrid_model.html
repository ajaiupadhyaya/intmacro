import numpy as np
import matplotlib.pyplot as plt
from ipywidgets import interact, FloatSlider

def solow_romer_model(s=0.2, delta_k=0.05, alpha=0.33, delta_A=0.1, l_A=0.2, L=100, A0=1, K0=10, T=50):
    T = int(T)
    A = np.zeros(T)
    K = np.zeros(T)
    Y = np.zeros(T)

    A[0] = A0
    K[0] = K0

    L_A = l_A * L
    L_Y = (1 - l_A) * L

    for t in range(1, T):
        A[t] = A[t-1] + delta_A * L_A * A[t-1]
        Y[t-1] = K[t-1]**alpha * (A[t-1] * L_Y)**(1 - alpha)
        K[t] = K[t-1] + s * Y[t-1] - delta_k * K[t-1]

    Y[-1] = K[-1]**alpha * (A[-1] * L_Y)**(1 - alpha)

    fig, axs = plt.subplots(1, 2, figsize=(14, 5))

    axs[0].plot(K, label='Capital K(t)', color='blue')
    axs[0].plot(A, label='Ideas A(t)', color='purple')
    axs[0].set_title("Capital and Knowledge Over Time")
    axs[0].set_xlabel("Time")
    axs[0].legend()
    axs[0].grid(True)

    axs[1].plot(Y, label='Output Y(t)', color='green')
    axs[1].set_title("Output Over Time")
    axs[1].set_xlabel("Time")
    axs[1].grid(True)
    axs[1].legend()

    plt.suptitle("ðŸ§¬ Solow-Romer Hybrid Growth Model", fontsize=14)
    plt.tight_layout()
    plt.show()

ðŸ§¬ Solow-Romer Hybrid Model#

This model combines:

  • Capital accumulation (from Solow)

  • Endogenous technology (from Romer)

Equations#

Output: [ Y = K^\alpha (A L_Y)^{1 - \alpha} ]

Capital accumulation: [ \frac{dK}{dt} = s Y - \delta K ]

Ideas: [ \frac{dA}{dt} = \delta_A L_A A ]

Labor is split:

  • ( L = L_Y + L_A )

  • Total labor grows exogenously at rate ( n )

ðŸ§  Key Insight#

In this model, both A(t) and K(t) grow â€” and so does Y(t).
There is no steady-state in levels â€” the whole system grows over time.

It captures:

  • Transition â†’ Solow style

  • Sustained growth â†’ Romer style
