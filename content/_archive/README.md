# Archive

Historical material preserved with provenance, per the project archive policy
(see `CONTRIBUTING.md`): nothing is deleted, superseded material moves here.
This directory is excluded from the Quarto render (`_quarto.yml`).

## Contents

- `*.ipynb` — v1 model notebooks superseded by canonical chapters. The merge
  map (which archive notebook fed which canonical chapter) lives in
  `content/MODELS.md`.
- `jupyter-lite.json` — JupyterLite app config from the v1 Jupyter Book era
  (moved here 2026-07-02 during the v2 Quarto cleanup; the deployed site no
  longer builds a JupyterLite workspace).
- `jupyterbook_custom.css` — theme overrides from the v1 Jupyter Book site,
  formerly `_static/custom.css` (moved here 2026-07-02; superseded by the
  `theme/*.scss` design system).
